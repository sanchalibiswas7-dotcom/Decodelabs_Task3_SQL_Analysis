# SQL Data Analysis on E-Commerce Dataset

## Objective

Analyze E-commerce transaction data using SQL queries to extract meaningful business insights.

## Tools Used

- Python
- Pandas
- SQLite3
- SQL
- Colab Notebook

## SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- COUNT()
- SUM()
- AVG()

## Key Insights

- The dataset contains 1,200 customer orders.
- Total revenue generated was ₹1,264,761.96.
- Chair generated the highest revenue of ₹195,620.11.
- Online Payment was the most frequently used payment method with 258 transactions.
- The average order value was ₹1,053.97.
